# Abacore

Abacore is a premium AI automation agency website built with Next.js App Router, TypeScript, Tailwind CSS, Framer Motion, shadcn-style UI primitives, and Lucide React icons.

## Features

- Mobile-first premium SaaS design
- Sticky responsive navigation with animated mobile menu
- High-conversion home page with hero, services, process, case studies, testimonials, and CTA
- Dedicated pages for Services, Pricing, About, and Contact
- SEO metadata, OpenGraph image route, robots, and sitemap
- Dark-mode-first styling with reusable components

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## Notes

- The contact form is wired for a polished front-end experience and can be connected to a backend or automation platform.
- The calendly and social links are placeholders and should be replaced with the live business profiles when ready.
- Update the `metadataBase`, sitemap URLs, and social links with the final production domain before launch.
